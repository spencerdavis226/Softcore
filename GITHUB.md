**Alternate longer (~350 characters, for longer bio fields or social cards if needed):**

Softcore is a Retail World of Warcraft Lua addon for hardcore-style leveling with friends. It tracks per-character runs, deaths, rule violations, party compatibility, proposals, and audit history. Sync uses addon channel messages; each player controls their own character validity—remote state is display and compatibility, not silent overwrite.
